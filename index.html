CSS Transforms

Transforms change the shape and position of the affected content by modifying the coordinate space.

Transforms do not disrupt the normal document flow.

perspective

perspective length can be pixels/ems, and it represents the approximate distance of how far away you are from that transforming element.

You can add perspective in 2 places: either on the transform statement, or its parent element. Placing it on the parent element is preferred since any additional elements will also have the same perspective.

See the Pen CSS T&T - 3D transforms - perspective by A.V. ( @avcoder) on CodePen.

perspective

Lower values bring you closer to the object and distorted more. Higher values show a more flatter effect.

See the Pen CSS T&T - 3D transforms - perspective by A.V. ( @avcoder) on CodePen.

perspective-origin

Perspective-origin establishes the vanishing point which by default is 50% 50% (i.e. the center).

Try setting the perspective origin to 50% 100%. Then 5% 100%.

See the Pen CSS T&T - 3D transforms - translateZ by A.V. ( @avcoder) on CodePen.

transform-style

Determines how child elements are treated in 3D space.

If (the default) transform-style: preserve-3d is set, then all transforms are applied to the children.

If transform-style: flat is set, then the elements appear to be in the 2D plane of the parent element
